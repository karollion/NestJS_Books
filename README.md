# Books: NestJS API server

## Description

NestJS server with books

Server run on `localhost:3030`
Data Base was in `mysql://localhost:3306/booksdb`

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Test

```bash
# unit tests
$ yarn run test

# e2e tests
$ yarn run test:e2e

# test coverage
$ yarn run test:cov
```
